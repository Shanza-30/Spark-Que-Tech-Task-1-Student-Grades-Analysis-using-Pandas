# 🎓 Student Grades Analysis using Pandas

## 📌 Overview

This project analyzes student performance using Python and Pandas.
It calculates average scores, identifies top performers, and visualizes data.

---

## 🎯 Objectives

* Load student dataset from CSV
* Calculate average score of each student
* Identify the top-performing student
* Analyze subject-wise performance
* Visualize data using graphs

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas 📊
* Matplotlib 📈
* Jupyter Notebook / Google Colab

---

## 📂 Dataset

The dataset contains the following columns:

* Name
* Math
* English
* Science

You can use a custom dataset or download from Kaggle.

---

## ⚙️ Steps Performed

1. Imported required libraries
2. Loaded dataset using Pandas
3. Checked missing values and data types
4. Calculated average score
5. Found top-performing student
6. Analyzed max and min scores
7. Calculated subject-wise averages
8. Created histogram for score distribution
9. Plotted bar chart for top students
10. Assigned grades (optional)

---

## 📊 Results

* Successfully calculated student averages
* Identified top-performing student
* Visualized performance using graphs
* Gained insights into subject-wise performance
